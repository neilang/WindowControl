h1. WindowControl

Safari extension that stops web pages from opening new windows without the navigation bar, status bar, hidden scroll bars etc. Also stops pages from resizing or moving windows.

h2. Download and Install

You can download the extension "here":https://github.com/downloads/neilang/WindowControl/WindowControl.safariextz. Double click the extension file to install it in safari. 

If you wish to remove it, go to your safari preferences > extensions tab > WindowControl > uninstall.
